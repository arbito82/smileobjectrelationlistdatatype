Tested with
-----------
4.5.0


Installation
------------

1. Copy the extension to the /extension folder, so that 
you have /extension/smileobjectrelationlistdatatype/* structure.

2. Enable the extension (either via administration panel or directly with 
settings files):
[ExtensionSettings]
ActiveExtensions[]=smileobjectrelationlistdatatype


3. Now you can use smilematrix datatype as information collector like any other datatype when editing classes.

4. Clear cache.
